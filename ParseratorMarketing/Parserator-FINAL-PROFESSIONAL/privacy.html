Privacy Policy

Last Updated: June 12, 2025

Overview

Parserator ("we", "our", or "us") is committed to protecting your privacy and pioneering Exoditical Moral Architecture (EMA) principles. This Privacy Policy explains how we collect, use, and protect information when you use our Chrome extension, API services, and website.

Information We Collect

  • API Usage Data: Request metadata including timestamps, response times, and error codes (no content stored)
  • Extension Settings: Your preferences and configuration stored locally in your browser
  • Parse Requests: Text content you choose to parse (processed in memory, never stored)
  • Account Information: Email address and API keys for registered users
  • Analytics Data: Anonymous usage patterns to improve service quality

How We Use Information

  • Process parsing requests and return structured data immediately
  • Improve parsing accuracy through aggregated pattern analysis
  • Provide customer support when explicitly requested
  • Generate anonymous usage analytics for service optimization
  • Ensure API rate limiting and prevent abuse

Data Storage and Security

  • No Content Storage: We do NOT store parsed content after processing
  • Temporary Processing: Data exists in memory only during active parsing
  • API Logs: Metadata retained for 30 days for debugging (no content)
  • Encryption: All data transmission uses HTTPS/TLS encryption
  • Infrastructure: Hosted on Google Cloud with enterprise security

Chrome Extension Permissions

Our extension requires minimal permissions:

  • activeTab: To read content from the current tab when you trigger parsing
  • storage: To save your preferences and API key locally
  • contextMenus: To add "Parse with Parserator" right-click option
  • sidePanel: To display parsing results in Chrome's side panel

We do NOT have permission to access your browsing history, bookmarks, or other tabs.

Third-Party Services

We minimize third-party dependencies:

  • Google Cloud Platform: API hosting and infrastructure
  • Firebase Hosting: Website delivery (no tracking)
  • No Analytics Trackers: We don't use Google Analytics, Facebook Pixel, etc.
  • No Ad Networks: We will never serve advertisements

Your Control Options

  • API Key Management: Revoke and regenerate keys anytime
  • Data Export: Request full data export via parse@parserator.com
  • Account Deletion: Complete removal within 24 hours
  • Opt-out: Disable all analytics in extension settings

Data Retention

  • Parsed Content: 0 days (never stored)
  • API Logs: 30 days (metadata only)
  • Account Data: Until deletion requested
  • Analytics: 90 days (anonymous, aggregated)

International Data

Our servers are located in the United States (Google Cloud US regions). By using Parserator, you consent to data processing in the US. We ensure appropriate safeguards for international data transfers.

Children's Privacy

Parserator is not intended for users under 13 years of age. We do not knowingly collect information from children under 13.

California Privacy Rights (CCPA)

California residents have additional rights:

  • Right to know what personal information is collected
  • Right to delete personal information

  • Right to non-discrimination for exercising privacy rights

For EU users, we ensure:
